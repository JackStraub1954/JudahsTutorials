Digression: The exclusive or (^) of two boolean values is true if exactly one of the operands is true: 

TRUE ^ FALSE  = TRUE
FALSE ^ TRUE  = TRUE
TRUE ^ TRUE   = FALSE
FALSE ^ FALSE = FALSE

In bitwise logic, 0 is taken to be FALSE and 1 is taken to be TRUE, so the bitwise-or of blue and medium gray will be: 

   0000 0000 1111
 ^ 0101 0101 0101
 ----------------
   0101 0101 1010
