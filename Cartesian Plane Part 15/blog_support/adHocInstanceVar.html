*About adHocInstanceVar: Recall that you cannot change the value of a local variable from an anonymous class; recall also that a lambda is an anonymous class. That makes code like this illegal: 

// This code won't compile!!!
private Color getFeedbackColor()
{
    Color color = null;
    GUIUtils.schedEDTAndWait( () -> 
        color = feedbackWindow.getBackground()
    );
    return color;
}

{
    GUIUtils.schedEDTAndWait( () -> 
        adHocInstanceVar= feedbackWindow.getBackground()
    );
    return adHocInstanceVar;
}
